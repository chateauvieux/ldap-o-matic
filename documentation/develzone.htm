MIG guts can be found here. These pages are technical documents concerning the PHP code used in MIG. HTML pages are not documented as far as they simply use MIG  objects' methods.

First thing to now : all PHP files are located in the php directory.

Second important thing : Attribute, search_class, user, and group objects inherits from a superclass "objectmodel" defining 4 properties, and 3 functions : 

path primary key for locating the object in the core
error_msg Last operation's error message
conn_id core connection identifier
count counts the number of elements of the derived object's lists
quit_on_success  
quit_on_error  
check_references  

The security file deals with web authentication when a user logs on MIG.
