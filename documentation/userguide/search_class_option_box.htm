To retrieve the search classes that the active user is authorized to browse, and display them in a HTML drop-down menu, simply insert the following line in your page body:

<script language="php">
require "php/class_option_box.php";
</script>

This will only work if the core layer has been connected and if the search class definition has been loaded.

This will create an HTML option box that will react on mouse events and will link to the same page, providing two PHP variables : 

dm, which is an array containing all search classes viewable by the active user;
selected_class, which is a search_class object correponding to the currently selected class.
