To create private pages, only available to MIG users, insert the following code at the top of your HTML pages, before the <HTML> tag. 

<script language="php">
require "php/config.php";
## security enabler
$auth_file = "php/" . AUTHENTICATION_SYSTEM_TYPE . "_authenticate.php";
require $auth_file;
</script>

As this code will open a PHP session, don't forget to provide a "logout" page containing :

<script language="php">
session_unset();
session_destroy();
</script>

For restricting the access to MIG administrators only, replace the above code by this :

<script language="php">
require "php/config.php";
## security enabler
$auth_file = "php/" . AUTHENTICATION_SYSTEM_TYPE . "_authenticate_manager.php";
require $auth_file;
</script>

And don't forget to close the session as well.

NOTE : this page requires a login page called login.html in your website. The file must contain the following form components:

UID<input type="text" name="uid">
Password<input type="password" name="password">

The form might point to your website main page.
