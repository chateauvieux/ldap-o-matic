Once the user/administrator is authenticated, a connection must be initiated with the core layer. To achieve this, simply insert the following code:

$core = new core();
if (($conn_id = $core->connect()) == FALSE) die($core->error_msg);

If an error occurs, the execution is aborted and an error message is displayed on the client's browser.

This requires that the core layer object has been already loaded. (see MIG's files section)
