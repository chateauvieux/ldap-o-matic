migauthorizedvalue - array of authorized values for the attribute

Migauthorizedvalues are indexed using integers starting with 0.

migauthorizedvalue[0] = value1
migauthorizedvalue[1] = value2

Values are free strings, with some restrictions :

inside the value, a '=' (equal) sign is treated as a <code,value> separator by MIG and processed by the process_migauthorizedvalues function.
the '#INCLUDE' and '#STRING' strings are MIG keywords processed by the process_migauthorizedvalues function.
if the attribute's is defined as a combination of attributes, the value must be either a valid attribute name or a string begining by the '#STRING' keyword.
