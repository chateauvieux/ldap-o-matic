migsearchtype - attribute's type property in a searching context

Currently supported types are :
date Date
dropdownmenu  Drop-down menu
dropdownmenumultiple  Drop-down multiple selection menu
dropdownmenuother  Drop-down menu with other choice
mail  Email address 
fourdigittext  Four-digit text
fifteendigittext  Fifteen-digit text
imageurl  Image URL
onelinetextbox  One-line text box
readonlydate Read-only date
readonlytext  Read-only text line
readonlyscrollingtextbox  Read-only scrolling text box
scrollingtextbox  Scrolling text box
twodigittext  Two-digit text
url  URL
combination  Combination of attributes)
