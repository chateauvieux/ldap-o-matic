process_migauthorizedvalues - process attribute's migauthorizedvalues

boolean process_migauthorizedvalues()
 

Process attribute migauthorizedvalue values in the following way :

split the value into two pieces, "name" and "value", in the migauthorizedvalue_couples attribute property. If a value contains the '=' (equal) symbol, the left part will be copied in  migauthorizedvalue_couples[i]["name"] and the right part in the migauthorizedvalue_couples[i]["value"]. Otherwise the value is copied as is in migauthorizedvalue_couples[i]["name"] and  migauthorizedvalue_couples[i]["value"].
if a value begins by the '#' symbol, then :
if this value is #INCLUDE, the function will import the file whose path is given after this keyword, and process them. The internal migauthorizedvalue counter is increased by the number of processed lines. Not that this keyword is not recursive, so a file cannot contain a #INCLUDE keyword. In the other hand, a file can contain <XXXX=YYYY> lines which will be processed automatically.
if this value if #STRING, then function will copy the value given after this keyword.

Example : let us consider the following attribute having these four migauthorizedvalues : 

My First Value
Second=My Second Value
#STRING My Third Value
#INCLUDE "php/my_include_file.inc"

and the php/my_include_file.inc containing the following :

A value
2=Other value

A call of the process_migauthorizedvalues will :

set the migauthorizedvalue property to :
migauthorizedvalue[0] = My First Value
migauthorizedvalue[1] = Second=My Second Value
migauthorizedvalue[2] = My Third Value
migauthorizedvalue[3] = A value
migauthorizedvalue[4] = 2=Other value
set the migauthorizedvalue_couples property to :
migauthorizedvalue_couples[0]["name"] = My First Value
migauthorizedvalue_couples[0]["value"] = My First Value
migauthorizedvalue_couples[1]["name"] = Second
migauthorizedvalue_couples[1]["value"] = My Second Value
migauthorizedvalue_couples[2]["name"] = My Third Value
migauthorizedvalue_couples[2]["value"] = My Third Value
migauthorizedvalue_couples[3]["name"] = A value
migauthorizedvalue_couples[3]["value"] = A value
migauthorizedvalue_couples[4]["name"] = 2
migauthorizedvalue_couples[4]["value"] = Other value

Note that the migauthorizedvalue counter was increased to 5.

This function returns false if an error was encountered (bad include file name or bad keyword), true otherwise.
