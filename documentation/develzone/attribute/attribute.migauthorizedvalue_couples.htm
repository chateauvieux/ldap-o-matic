migauthorizedvalue_couples - array of <code,value> of the attribute's migauthorizedvalues

Migauthorizedvalue couples are indexed using integers starting with 0.

migauthorizedvalue_couples[0]["name"] = name1
migauthorizedvalue_couples[0]["value"] = value1
migauthorizedvalue_couples[1]["name"] = name2
migauthorizedvalue_couples[1]["value"] = value2

This property is only available after calling the process_migauthorizedvalues function.
