delete - deletes the attribute of MIG schema

boolean delete(string [type])
 

If type is specified and is equal to "system", the system attribute and all user's preferences for this attribute will be deleted.

If type is specified and is *not* equal to "system", no referency check will be done during delete operation. Usefull when you want to delete a user's preference attribute.

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
