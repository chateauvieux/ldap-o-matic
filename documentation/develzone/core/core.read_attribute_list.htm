read_attribute_list - reads a bunch of attributes on  the information system

array read_attribute_list(string cmd, array [result_array], string [string])

Many different responses are obtained depending on the choice of specific or not parameters in the function call :

  cmd result_array string function
1 user [array] < none > returns current user's search preferences. If array is specified, restricts result to settings listed in array
2 system [array] < none > returns system search preferences. If array is specified, restricts result to settings listed in array
3 both [array] < none > returns (2) + (3)
4 system array class returns system search preferences for class search class. Limits results to array settings.
5 user array class returns current user's search preferences for class search class. Limits results to array settings.
6 referencies array class Retrieve the list of attributes that are linked to a search class.
7 combinations array attribute Retrieve the list of attributes that are pointing to the one specified.

The sixth scenario is useful if the information system does not hold referencies between attributes and search classes, such as that an attribute is a relevant attribute for a search class, or a default search criteria for this search class.

The seventh scenario is also useful if the information system does not hold referencies between attributes. 

This function returns the attributes array on success, false otherwise. In the later, more information can be displayed using the error_msg information.
