mod_add - updates information by adding field/value couples.

boolean add(string dn, array how) 
 

The array how specifies the information to add in the entry. The values in the entries are indexed by individual attributes. In case of multiple values for an attribute, they are indexed using integers starting with 0.

what["attribute1"] = value
what["attribute2"][0] = value1
what["attribute2"][1] = value2

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
