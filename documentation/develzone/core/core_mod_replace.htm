mod_replace - updates information by replacing attribute/value couples

boolean mod_replace(string dn, array how) 
 

The array how specifies the information to update in the entry. The values in the entries are indexed by individual attributes. In case of multiple values for an attribute, they are indexed using integers starting with 0.

what["attribute1"] = value
what["attribute2"][0] = value1
what["attribute2"][1] = value2

An attribute having a unique value that equals to an empty string will be removed from the array.

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
