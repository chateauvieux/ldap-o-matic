connect - initiate a connection with the information system

int connect(string [uid], string [pwd] )

In normal operation, uid and pwd are not specified, and the connection can only be initiated after a successful authentication of the user done by the security layer. It will be achieved :

thanks to administrator's username and password account as defined in the configuration file (AUTH_UID constant) if the authenticated user is a MIG administrator,
with user's username and password (provided during authentication) if the authenticated user is an ordinary MIG .

In some particular cases, a connection can be initiated manually by specifying uid and pwd.

This function returns the connection identifier on success, false otherwise. In the later, more information can be displayed using the error_msg information.
