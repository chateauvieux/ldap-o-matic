override_config - overridde preferences

boolean override_config(array config, array user_config, array attributes)

When calling this function, the config array must contain a list of attributes to be overridden by the user_config array, also containing attributes. Typically config will contain system settings and user_config, user settings. The attributes array specifies which attributes are to be overridden (typically : migsearchcriteriaclass, migsearchresultclass and default_criteria).

The overridden result array is the config array.

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
