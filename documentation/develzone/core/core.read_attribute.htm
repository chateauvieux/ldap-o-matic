read_attribute - reads an attribute on  the information system

boolean read_attribute(string attribute, string type, array [result_array])

String may be "system" or "user", which will select respectively system or current user's attribute attribute. Attribute is a migattributename. 

The result_array parameter, optional, can be added to restrict the properties values returned by the server to just those required. This is much more efficient than the default action (which is to return all properties values). The use of this parameter should therefore be considered good practice.

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
