read_search_class_list - reads a bunch of search classes on  the information system

array read_search_class_list(string cmd, array [result_array], string [search_class])

Many different responses are obtained depending on the choice of specific or not parameters in the function call :

  cmd result_array class function
1 auto [array] < none > returns current user's viewable search classes. If the user is an administrator all search classes are returned. If array is specified, restricts result to search class properties listed in array
2 referencies array search_class returns search classes linked to search preferences. If array is specified, restricts result to search class properties listed in array

This function is secure : the list of search classes returned when in (1) depends on user grants.

The second scenario is useful if the information system does not hold referencies between attributes and search classes, such as that a search class is set as a search criteria for an attribute.

This function returns a search class on success, false otherwise. In the later, more information can be displayed using the error_msg information.
