read_user_from_list  - reads a user from a user array

boolean read_user_from_list(array user_list, var which)

Retrieves the search user indexed by which from the user_list array.

The which variable can be of two types. If an integer is passed, the retrieved user will be the one indexed by which in the array. If a string is passed, the retrieved user will be the one having which as ID (i.e, as uid)

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
