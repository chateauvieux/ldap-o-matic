mod_del - Deletes search class properties

boolean mod_del(array how)
 

The array how specifies the search class' properties to be deleted. The values of the properties are indexed by individual names. In case of multiple values, properties are indexed using integers starting with 0.

how["property1"] = value
how["property2"][0] = value1
how["property2"][1] = value2

This function returns true on success, false otherwise. In the later, more information can be displayed using the error_msg information.
