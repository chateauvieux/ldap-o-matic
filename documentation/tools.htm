Provided tools to manipulate the objects

Attributes administration tool

The Attributes administration tool allow an administrator to create, edit or delete attributes available in the MIG system. When an attribute is created, the administrator has to provide the pointed low-level attribute attribute (i.e, the directory attribute) and a Display name (that will appear in the web interface). Additional details can be supplied as well : 

the options of an attribute, that is to say the different set of values the attribute can have;
the "type" of the attribute in different use contexts (that is to say, the manner the attribute will be displayed in web pages depending on each situation, which can be a viewing, editing or searching context);
the attribute description (useful in help pages);
some examples of attribute values (useful in help pages).

Search class administration tool

The Attributes administration tool allow an administrator to create, edit or delete search classes available in the MIG system. When a search class is created, the administrator must set the class Display name (that will appear in the web interface), and some additional information, such as the subset of attributes composing the search class, and the location of the data itself (in other words, a LDAP search base, a database table name, etc). Data can be filtered by type filters (by the "objectclass" attribute value in LDAP for example). 

Each search class may be public or private, to control information access over your information systems.

Users administration tool

The Users administration tool allow an administrator to edit existing users’ grants (user, administrator, or none) and properties for MIG using purpose. Here he can define which search classes each user is allowed to access.

 

Groups administration tool

The groups administration tool allow an administrator to create, edit or delete groups, that give power to existing users’ to maintain directory specific directory data subsets. Those data subset are definable per attribute, thanks to MIG search classes.
